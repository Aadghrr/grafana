Used to align multiple inline form fields in one row. The row will wrap if the width of the children exceeds its own. Equivalent to the div with `gf-form-inline` class name.

### Usage

```jsx
<InlineFieldRow>
  <InlineField label="Label Row 1">
    <Input placeholder="Label" />
  </InlineField>
  <InlineField label="Label Row 1">
    <Input placeholder="Label" />
  </InlineField>
</InlineFieldRow>
```
